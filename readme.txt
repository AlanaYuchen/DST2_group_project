To run our project, please do as follows:

1. Create a mySQL database with name 'sys'.

2. Create tables by running codes in /src/main/resources/schema.sql;

3. Change url, username and password in /src/main/resources/app.properties;

3. Run codes in /src/main/java/cn.edu.zju/cmd/PharmGKBImporter.java;

4. Configure Tomcat environment;
